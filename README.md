# A2G Smart Notes 2.0 - Starter Scaffold
This repository contains a ready-to-edit Next.js 15 scaffold with Firebase integration ready.

**What this scaffold includes**
- Next.js 15 app router starter
- Tailwind CSS + shadcn layout
- Firebase client setup (Auth & Firestore)
- Basic pages: Home, Login, Dashboard, Admin (notes upload), MCQ practice
- Admin-only example guard
- Scripts to build & start

**Next steps**
1. Connect this repo to Firebase Hosting (Console → Hosting → Connect to GitHub → branch `main`).
2. In Firebase Console, add your Firebase config values to `.env.local`.
3. Install dependencies locally: `npm install`
4. Build & test: `npm run build && npm start`
5. Commit & push changes to trigger Firebase Hosting deploy.
